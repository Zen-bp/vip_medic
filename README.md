 ### vip_medic

| Param name    | Description                    |
| ------------- | ------------------------------ |
| `medic`       | 0/1 - enable/disable feature   |
| `medic_limit` | use limitation per round (0 - unlimited)     |
| `medic_delay` | Delay between uses (float)   |
| `medic_hp`    | heal hp amount   |
